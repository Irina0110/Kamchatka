## Installation

1. Выполните `git clone` репозитория
   ```sh
   git clone https://github.com/Irina0110/Kamchatka
   ```
2. Установите зависимости
    ```sh
   npm install
    ```
3. Запустите проект
    ```sh
   npm run dev
    ```
